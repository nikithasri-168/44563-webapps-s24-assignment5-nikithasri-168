Q) Which row breaks first? The letters or the drink names.

Ans: The drink names row ("Gin, Apple Juice, Red wine, Apple cider, Pepsi, Almond milk") will break first as the screen size is gradually decreased since they are set to adapt to medium-sized displays (col-md medium devices - screen width equal to or greater than 768px). Only when the screen size is severely lowered (col-sm-2 small devices - screen width equal to or greater than 576px) will the letter row wrap to the following row and continue to horizontally contract. This is a result of the responsive nature of the Bootstrap grid framework.
